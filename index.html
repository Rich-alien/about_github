Установка гита - git init "создается репозиторий"

создание документа для описания - echo >> README.md

установка настроек ,в данном случае локальном - git config --local (--global для глобального) user.name "Salavat" (писать отдельно) user.email richalien@yandex.ru

заставляем гит следить за файлами - git add -A (-A все файлы)

просмотр статуса репозитория- git status

запомнить эти изменения - git commit -a(все файлы) -m(добавляем сообщение) "краткая инфа , что в изменениях"

проверка логов коммитов- git log

красивый вывод логов - git log --pretty=format:"%h %ad | %s%d [%an]" --graph --date=short

документация- git log --help

добавление репазитория в наш гит git remote (обозн что мы работаем с удаленным репозиторием) add(добавляем репозиторий) origin(псевдоним нашего репозитория , называть можно как угодно) https://github.com/Rich-alien/about_github.git (путь)

git push (перемещение с локального в github) -u(куда по умолчанию пушить наши файлы) test(название) master(наша главная ветка)
